# 新闻

欢迎来到 **GEEKFUN 新闻！** :rocket::rocket::rocket:

及时了解 GEEKFUN 社区的最新公告、发布和更新。在这里您将找到关于我们项目的新闻、新功能、社区活动和重要里程碑。

## [🎉 DocKit v0.9.10 发布：跨账户导入、变量支持与 DSL 自动补全](https://github.com/geek-fun/dockit/releases/tag/v0.9.10)

**2026 年 4 月 28 日** - DocKit v0.9.10 带来跨账户数据导入（JSON、CSV、S3、DynamoDB 表）、请求集合中的 Profile/Table ARN 变量支持，以及变量自动补全和语法高亮。

## [🎉 DocKit v0.9.9 发布：DynamoDB SSO/AssumeRole 认证与 DSL 自动补全](https://github.com/geek-fun/dockit/releases/tag/v0.9.9)

**2026 年 4 月 28 日** - DocKit v0.9.9 为 DynamoDB 连接加入 SSO 和 AssumeRole 认证、AWS 配置文件和环境变量认证、基于凭证的多表连接模型、统一可搜索选择组件、Search DSL 自动补全改进，以及 rustls-webpki 安全漏洞修复。

## [🎉 DocKit v0.9.8 发布：键盘导航与历史视图无障碍改进](https://github.com/geek-fun/dockit/releases/tag/v0.9.8)

**2026年4月22日** - DocKit v0.9.8 正式发布。此补丁版本为历史视图加入了键盘导航和更清晰的空状态引导，进一步提升最近查询工作流的可访问性与易用性。

## [🎉 DocKit v0.9.7 发布：连接克隆、查询自动补全与导入增强](https://github.com/geek-fun/dockit/releases/tag/v0.9.7)

**2026年4月20日** - DocKit v0.9.7 新增连接克隆、查询参数值自动补全，以及导入时创建新索引/新表的能力，进一步提升连接配置和数据导入效率。

## [🎉 DocKit v0.9.6 发布：修复更新安装问题](https://github.com/geek-fun/dockit/releases/tag/v0.9.6)

**2026年4月14日** - DocKit v0.9.6 修复了更新安装失败的问题，通过在下载前重新获取更新清单，避免因签名 URL 过期导致安装中断。

## [🎉 DocKit v0.9.5 发布：强化 CI 与发布流水线](https://github.com/geek-fun/dockit/releases/tag/v0.9.5)

**2026年4月14日** - DocKit v0.9.5 强化了发布流程：在推送到 `master` 时也执行 CI 校验，并要求所有平台构建成功后才允许正式发布。

## [🎉 DocKit v0.9.4 发布：连接排序筛选与更新进度显示](https://github.com/geek-fun/dockit/releases/tag/v0.9.4)

**2026年4月14日** - DocKit v0.9.4 为连接列表加入排序与筛选，新增更新下载/安装进度条，并修复了 Windows 上的快捷键问题。

## [🎉 DocKit v0.9.3 发布](https://github.com/geek-fun/dockit/releases/tag/v0.9.3)

**2026年4月8日** - DocKit v0.9.3 已通过 GitHub Releases 发布。这是一个维护版本，提供了适用于各桌面平台的最新安装包。

## [🎉 DocKit v0.9.2 发布](https://github.com/geek-fun/dockit/releases/tag/v0.9.2)

**2026年4月6日** - DocKit v0.9.2 作为 v0.9 系列的又一个维护版本已在 GitHub Releases 提供，包含最新桌面构建产物。

## [🎉 DocKit v0.9.1 发布：Elasticsearch API Key 认证与自动补全修复](https://github.com/geek-fun/dockit/releases/tag/v0.9.1)

**2026年3月25日** - DocKit v0.9.1 为 Elasticsearch 连接加入 API Key 认证，并修复自动补全问题，让首个 v0.9 补丁版本在安全集群场景下更加可靠。

## [🎉 DocKit v0.9 发布：UI 重设计、查询历史与 DynamoDB Local 支持](/zh/news/dockit-v0-9-release)

**2026年3月3日** - DocKit v0.9 正式发布！此版本带来全面的 UI 重设计（shadcn-vue + UnoCSS）、Elasticsearch 和 DynamoDB 的自动查询历史、DynamoDB Local 端点支持以及 PartiQL 文档格式化功能。了解这一里程碑版本的全部新内容。

**2026年1月20日** - 我们很高兴地宣布 DocKit v0.8 正式发布，完整支持 Amazon DynamoDB！此版本带来了强大的新功能，包括 PartiQL 查询编辑器、可视化查询 UI、内联数据编辑等。了解这次重大更新的更多新特性。
