# 下载 DocKit
感谢您尝试使用 DocKit！:rocket::rocket::rocket: DocKit 是一个全新的 NoSQL GUI 客户端，具有超轻量级并支持 Windows、macOS 和 Linux 平台。
DocKit 支持 Elasticsearch、OpenSearch、[DynamoDB](https://aws.amazon.com/dynamodb/)，更多支持即将到来（如 [MongoDB](https://www.mongodb.com/) 等），您可以从以下链接下载最新版本的 DocKit。

## 🎉 v0.8 版本新特性

**全面支持 DynamoDB！**

DocKit v0.8 版本带来了完整的 Amazon DynamoDB 支持：
- ✅ **可视化查询界面** - 无需编写代码即可扫描和查询表
- ✅ **PartiQL 查询编辑器** - 类 SQL 语法，智能自动补全
- ✅ **内联数据编辑** - 直接从查询结果更新和删除数据项
- ✅ **简单项目创建** - 通过 UI 表单创建新数据项
- ✅ **脚本持久化** - 自动保存 PartiQL 查询到本地

**为什么选择 DocKit 管理 DynamoDB？**
- 🌟 **完全开源** - 社区驱动，持续改进
- ⚡ **更高效** - 桌面应用，无需容器或服务器
- 🎯 **多数据库支持** - 在一个界面管理 DynamoDB、Elasticsearch 和 OpenSearch
