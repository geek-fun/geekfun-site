# 文档

DocKit 是一款 **AI 原生** NoSQL GUI 客户端，支持 MongoDB、Elasticsearch、OpenSearch、DynamoDB 和 EasySearch。用自然语言写查询，让 AI 处理语法。Agentic Data Studio 提供多步骤数据库操作，支持按源权限控制和安全门禁。

## [Agentic Data Studio](/zh/docs/dockit/agentic-datastudio)

DocKit 的核心功能。用自然语言描述需求——AI Agent 编写查询、检查表结构、更新文档并返回结果。包含权限模式、风险等级、确认规则和上下文压缩。支持 OpenAI、Anthropic、DeepSeek、Ollama、LM Studio 等多种 AI 提供商。

## [安装指南](/zh/docs/dockit/installation)

感谢您尝试使用 DocKit！:rocket::rocket::rocket: 本指南将帮助您开始使用 DocKit。

## [连接到服务器](/zh/docs/dockit/connect-to-server)

DocKit 支持连接 MongoDB、Elasticsearch、OpenSearch 和 DynamoDB。支持基础认证、API Key、SCRAM、AWS SSO、AWS Profile 和完整 URI 等多种认证方式。所有连接类型均支持 SSL/TLS。

## [设置](/zh/docs/dockit/settings)

配置 DocKit 以匹配您的工作习惯：主题（深色/浅色/自动）、语言（中文/英文）、编辑器选项（字体大小、缩略图、行号、Tab 大小、插入空格）和查询历史容量。AI 提供商支持代理配置。

## [快捷键介绍](/zh/docs/dockit/shortcut)

作为一个桌面应用程序，DocKit 提供了一组快捷键，帮助用户更高效地使用 DocKit。本文档概述了可用的快捷键及其对应的操作。随时按 `Cmd+?` 在应用内查看所有快捷键。

## [管理 Elasticsearch 集群](/zh/docs/dockit/manage-elasticsearch-cluster)

DocKit 支持监控和管理 Elasticsearch，包括索引和别名管理、节点管理、分片管理（含详细分片级指标）、模板管理（可组合模板、组件模板和传统模板）。

## [导入与导出](/zh/docs/dockit/import-export)

DocKit 支持以 JSON、CSV、JSONL 和 Elasticsearch Bulk 格式从 MongoDB、Elasticsearch、OpenSearch 和 DynamoDB 导入和导出数据。多步向导包含 Schema 映射、字段筛选、元数据导出和后台任务执行，支持实时进度追踪。

## [查询历史](/zh/docs/dockit/query-history)

永不丢失查询记录。DocKit 自动记录 MongoDB、Elasticsearch、OpenSearch 和 DynamoDB 的每个已执行查询。支持星标收藏、按路径或内容搜索、查看条目详情，以及重新运行或复制历史查询。

## [DynamoDB 管理与指标](/zh/docs/dockit/dynamodb-manage-metrics)

DocKit 允许您管理 DynamoDB 表并监控关键指标 — 表大小、项目数量、读/写容量 — 全部通过简洁的桌面 GUI 完成。完整的表生命周期管理，包括 GSI/LSI 管理、TTL、流、PITR、加密和 CloudWatch 监控指标。
