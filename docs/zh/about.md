# 关于我们

GEEKFUN 是一个致力于开源的俱乐部，我们专注于开发具有价值且可持续的软件。我们的使命是创造对世界产生积极影响的解决方案，同时确保其长期的可行性和相关性。
我们非常感谢您对我们项目的支持和贡献，它们帮助我们不断建设一个更好、更可持续的未来。

## 关注我们
如果您也同我们一样热爱开源，对我们的项目感兴趣，欢迎关注我们的社交媒体账号，获取最新的动态和更新。
