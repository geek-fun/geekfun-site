# DocKit v1.1 发布

DocKit v1.1 是自 v1.0 以来最大的更新，新增 **MongoDB** 和 **EasySearch** 支持、**Agentic Data Studio** 含 28+ 代理工具，以及全面的功能改进。

**v1.1 新功能：**

- **MongoDB** — 完整的连接、查询、管理、导入导出和 Agentic Data Studio 集成
- **EasySearch** — 现与 Elasticsearch 和 OpenSearch 一同受支持
- **Agentic Data Studio** — 跨 5 种数据库的 28+ 代理工具。权限模式（询问/自动）、风险等级（安全/升高/破坏性）、按源权限和确认规则
- **12+ AI 提供商** — OpenAI、Anthropic、DeepSeek、OpenRouter、Gemini、Grok、Mistral、Azure、Ollama、LM Studio 及自定义端点
- **AWS SSO 和 Profile 认证** — IAM Identity Center 设备认证和 AWS 配置文件支持
- **DynamoDB 表生命周期** — 4 步创建向导、GSI/LSI 管理、TTL、PITR、流、加密、CloudWatch 指标、清空
- **导入导出向导** — 多步操作，含 schema 映射、字段筛选、元数据导出和后台任务执行
- **后台任务** — 在后台运行导入导出，实时查看进度
- **文件浏览器** — 浏览和管理本地 `.search`、`.partiql`、`.mongo` 文件
- **查询历史** — 星标收藏、条目详情视图、按路径/内容搜索
- **侧边栏助手** — 快速问答的轻量级聊天面板，独立于 Data Studio
- **Elasticsearch 集群管理** — 分片级详细指标、模板类型、别名管理
- **连接克隆** — 一键复制任意连接

查看完整更新日志：[GitHub 发布](https://github.com/geek-fun/dockit/releases)
