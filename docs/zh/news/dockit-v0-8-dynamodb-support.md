# 🎉 DocKit v0.8 发布：全面支持 DynamoDB

**2026年1月20日** - 我们很高兴地宣布 DocKit v0.8 正式发布！这个版本为开源桌面 GUI 客户端带来了对 **Amazon DynamoDB** 的完整支持，进一步扩展了 DocKit 作为统一 NoSQL 数据库管理工具的能力。

## 🚀 新增功能亮点

**DynamoDB 核心功能**
- ✅ **可视化查询 UI** - 无需编写代码即可扫描和查询表，支持主键过滤和高级条件
- ✅ **PartiQL 查询编辑器** - 使用类 SQL 语法管理 DynamoDB，配备智能自动完成
- ✅ **内联数据编辑** - 直接从查询结果更新和删除项目
- ✅ **简单的项目创建** - 通过 UI 表单创建新项目，无需 JSON 模板
- ✅ **脚本持久化** - 自动保存 PartiQL 查询到本地文件系统

**统一管理体验**
- 🔗 在单一界面中同时连接多个 DynamoDB、Elasticsearch 和 OpenSearch 实例
- 💾 所有连接和查询自动持久化，永不丢失工作成果
- 🖥️ 跨平台支持：Mac、Windows、Linux

## 为什么选择 DocKit？

相比 AWS 控制台的网页界面容易误操作，以及现有桌面客户端（如 dynobase、dynomate）的闭源和缺乏维护，DocKit 提供：
- 🌟 **完全开源** - 社区驱动，持续改进
- ⚡ **更高效** - 桌面应用，无需容器或服务器
- 🎯 **多数据库支持** - 一个工具管理所有 NoSQL 数据库
