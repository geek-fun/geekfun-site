# News

Welcome to **GEEKFUN News!** :rocket::rocket::rocket:

Stay updated with the latest announcements, releases, and updates from the GEEKFUN community. Here you'll find news about our projects, new features, community events, and important milestones.

## [🎉 DocKit v0.9.10 Released: Cross-Account Import, Variable Support & DSL Autocomplete](https://github.com/geek-fun/dockit/releases/tag/v0.9.10)

**April 28, 2026** - DocKit v0.9.10 brings cross-account data import (JSON, CSV, S3, DynamoDB tables), variables in Profile and Table ARN fields, and variable autocomplete with syntax highlighting in request collections.

## [🎉 DocKit v0.9.9 Released: DynamoDB SSO, AssumeRole Auth & DSL Auto-completion](https://github.com/geek-fun/dockit/releases/tag/v0.9.9)

**April 28, 2026** - DocKit v0.9.9 adds SSO and AssumeRole authentication for DynamoDB connections, AWS profile and environment variable authentication, credential-based connection model for multi-table workflows, unified searchable select component, search DSL auto-completion improvements, and security fixes for rustls-webpki vulnerabilities.

## [🎉 DocKit v0.9.8 Released: Keyboard Navigation & History View Accessibility](https://github.com/geek-fun/dockit/releases/tag/v0.9.8)

**April 22, 2026** - DocKit v0.9.8 is now live. This patch release improves the history view with keyboard navigation and a clearer empty-state call to action, making recent-query workflows more accessible and easier to use.

## [🎉 DocKit v0.9.7 Released: Connection Cloning, Query Autocomplete & Import Improvements](https://github.com/geek-fun/dockit/releases/tag/v0.9.7)

**April 20, 2026** - DocKit v0.9.7 adds connection cloning, query parameter value autocomplete, and support for creating a new index or table during import. This release focuses on faster connection setup and smoother data workflows.

## [🎉 DocKit v0.9.6 Released: Updater Install Fix](https://github.com/geek-fun/dockit/releases/tag/v0.9.6)

**April 14, 2026** - DocKit v0.9.6 fixes updater installation failures by refreshing the update manifest before download, reducing issues caused by expired signed URLs.

## [🎉 DocKit v0.9.5 Released: CI & Release Pipeline Hardening](https://github.com/geek-fun/dockit/releases/tag/v0.9.5)

**April 14, 2026** - DocKit v0.9.5 strengthens the release process by gating CI checks on pushes to `master` and requiring all platform builds to succeed before publishing a release.

## [🎉 DocKit v0.9.4 Released: Connection Sorting, Filtering & Update Progress](https://github.com/geek-fun/dockit/releases/tag/v0.9.4)

**April 14, 2026** - DocKit v0.9.4 adds sort and filter controls for the connection list, introduces a progress bar for update downloads and installs, and fixes keyboard shortcuts on Windows.

## [🎉 DocKit v0.9.3 Released](https://github.com/geek-fun/dockit/releases/tag/v0.9.3)

**April 8, 2026** - DocKit v0.9.3 is available on GitHub Releases. This is a maintenance release with updated downloadable assets for all supported desktop platforms.

## [🎉 DocKit v0.9.2 Released](https://github.com/geek-fun/dockit/releases/tag/v0.9.2)

**April 6, 2026** - DocKit v0.9.2 is available on GitHub Releases as another maintenance update in the v0.9 line, with fresh desktop builds for supported platforms.

## [🎉 DocKit v0.9.1 Released: Elasticsearch API Key Auth & Autocomplete Fixes](https://github.com/geek-fun/dockit/releases/tag/v0.9.1)

**March 25, 2026** - DocKit v0.9.1 adds API Key authentication for Elasticsearch connections and fixes auto-completion issues, making the first v0.9 patch release more reliable for secured clusters.

## [🎉 DocKit v0.9 Released: UI Redesign, Query History & DynamoDB Local Support](/news/dockit-v0-9-release)

**March 3, 2026** - DocKit v0.9 is here! This release ships a complete UI redesign with shadcn-vue + UnoCSS, automatic query history for Elasticsearch and DynamoDB, DynamoDB Local endpoint support, and PartiQL document formatting. Read about everything new in this landmark release.

## [🎉 DocKit v0.8 Released: Full DynamoDB Support](/news/dockit-v0-8-dynamodb-support)

**January 20, 2026** - We're excited to announce DocKit v0.8 with complete Amazon DynamoDB support! This release brings powerful new features including PartiQL query editor, visual query UI, inline data editing, and more. Read more about what's new in this major update.
