# Document

DocKit as a NoSQL/NewSQL GUI client supports Elasticsearch, OpenSearch, DynamoDB, and more to come. This document outlines how to use DocKit.

## [Installation Guide](/docs/dockit/installation)

Thanks for trying out DocKit! :rocket::rocket::rocket: This guide will help you get started with DocKit.

## [Connect to Server](/docs/dockit/connect-to-server)

DocKit as a NoSQL GUI client, it supports multiple databases, including Elasticsearch, OpenSearch, DynamoDB, and more to come.
This document outlines how to connect to a database server using DocKit.

## [Shortcuts Introduction](/docs/dockit/shortcut)

As a Desktop Application, DocKit provides a set of shortcuts to help users use DocKit more efficiently. This document
outlines the available shortcuts and corresponding actions.

## [Manage Elasticsearch Cluster](/docs/dockit/manage-elasticsearch-cluster)

DocKit as a NoSQL GUI client, it supports to monitor and manage Elasticsearch including indices and alias management,
nodes management, shards management, and templates management. This document outlines how to monitor and manage
Elasticsearch in DocKit.
